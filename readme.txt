== Description ==
Serin es un tema moderno, responsivo y ligero desarrollado para sitios institucionales y académicos. 
Incluye soporte para bloques, cabecera personalizada, estilos accesibles y compatibilidad con plugins personalizados. 

== Installation ==
1. Sube la carpeta del tema a `/wp-content/themes/`.
2. Activa el tema desde el panel de administración de WordPress, en **Apariencia → Temas**.
3. Si es necesario, instala el plugin recomendado `Serin CPT` (para los tipos de contenido personalizados).
